## ðŸŒ GitHub Pages Deployment Instructions

Follow these steps to deploy this landing page to GitHub Pages:

### Step 1: Create a New Repository
1. Log in to your [GitHub Account](https://github.com/).
2. Click the **+** icon in the top-right corner and select **New repository**.
3. Name your repository (e.g., `technonova-landing-page`).
4. Ensure "Public" is selected.
5. Click **Create repository**.

### Step 2: Upload Files
1. In your new repository, click on **uploading an existing file**.
2. Drag and drop the `index.html` file from your `Task-2` folder.
3. Commit the changes by clicking **Commit changes**.

### Step 3: Enable GitHub Pages
1. Go to the **Settings** tab of your repository.
2. In the left sidebar, click on **Pages**.
3. Under **Build and deployment** > **Source**, select **Deploy from a branch**.
4. Under **Branch**, select `main` (or `master`) and `/ (root)`.
5. Click **Save**.

### Step 4: Access Your Live Site
- Refresh the page after a minute.
- You will see your live site URL at the top of the page.
- **Expected URL Format:** `https://your-username.github.io/technonova-landing-page/`
